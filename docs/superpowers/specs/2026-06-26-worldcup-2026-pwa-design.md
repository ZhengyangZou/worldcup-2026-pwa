# 2026 世界杯 PWA 设计规格

## 目标

构建一个个人和亲友使用的 2026 世界杯数据 PWA，手机和 PC 都可访问。界面风格参考懂球帝式体育资讯/比分 App：绿色顶部导航、紧凑比分列表、新闻流、数据榜单、可点击比赛/球队/球员详情。

## 技术与分发

- 使用 React + Vite + TypeScript 构建正式版。
- 做成 PWA，支持手机添加到主屏幕。
- 数据优先客户端直连 ESPN 等公开接口，失败时使用本地缓存和内置快照。
- 发布采用 GitHub Pages 作为国际/备份入口，另准备内地镜像链接。
- 现有 `worldcup-2026-ui-demo.html` 是 UI 原型参考，不作为正式工程代码直接迁移。

## 核心页面与模块

- 首页保留较完整的数据密度，不刻意瘦身。
- 顶部导航包含：首页、赛程比分、分组积分、淘汰赛、球员榜、球队。
- 新闻模块保留，展示标题、来源和外链；正式版可在可访问且合规的前提下抓取摘要/正文缓存，但必须保留原文链接。
- 赛程与比分模块展示今日/近期比赛，每场比赛包含：
  - 开球时间、场馆、状态。
  - 双方球队、国旗、比分。
  - 点击比赛主体进入赛况详情。
  - 每场比赛内显示直播或集锦入口。
- 赛程与比分模块内增加“淘汰赛对阵图”入口和对阵图预览。

## 比赛详情

比赛详情使用弹窗或详情页，包含以下 Tab：

- 赛况
- 阵容
- 数据
- 集锦
- 新闻

直播和集锦入口按比赛展示：

- 未开赛/进行中：澳视体育直播、央视频直播。
- 已结束：FIFA YouTube 集锦、央视频集锦。
- 有具体比赛直播间或具体视频链接时优先使用具体链接；没有时回退到频道或专题页。
- 显示提示：直播和集锦以当地版权可用性为准。

## 分组积分与晋级

- 正式版必须展示 12 个小组 A-L。
- 每组展示 4 队排名、国旗、比赛、净胜球、积分、晋级状态。
- 每组前 2 直接晋级。
- 每组第 3 进入“小组第三名排名”。
- 第三名排名展示 12 队，前 8 晋级。
- 第三名排序规则需完整实现：积分、净胜球、进球数、公平竞赛/纪律分，以及规则要求的兜底排序。
- 不增加“只看我的关注球队”或“只看晋级形势”筛选。

## 球员榜与资料

- 球员榜顶部保留可切换 Tab：射手榜、助攻榜、扑救榜、黄牌榜。
- 当前选中的榜单展示 Top 20，不在一个榜单下混放其他榜单摘要。
- 球员行展示排名、头像、姓名、国家队、位置、数据值。
- 点击球员进入球员信息。
- 球员头像不要求 100% 补齐：
  - 优先使用公开数据源头像。
  - 缺失时使用国家队色块 + 姓名首字兜底。
  - 可通过 `playerOverrides` 手动补充头像、中文名、俱乐部等字段。

## 球队资料

- 所有球队名尽量带国旗。
- 点击球队名进入球队信息。
- 球队信息展示小组排名、积分、晋级状态、赛程、核心球员和相关新闻入口。

## 明确不做

- 不做用户账号系统。
- 不做亲友竞猜或云同步。
- 不做首次打开时的地区设置；直播链接同时给港澳和内地入口，由用户自行选择。
- 不依赖单一 GitHub Pages 链接作为唯一分发方式。
