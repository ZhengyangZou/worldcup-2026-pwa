# 2026 世界杯数据中心

个人和亲友使用的轻量 PWA，用来查看 2026 世界杯赛程比分、分组积分、球员榜、新闻外链和每场比赛的直播/集锦入口。

## 当前范围

- 赛程与比分：打开 App 后自动从 ESPN 拉取比赛、比分、状态和球场信息。
- 比赛详情：从比赛卡片跳转到对应详情卡，保留赛况、阵容、数据、集锦、新闻 Tab。
- 分组积分：展示 12 个小组和小组第三名排名。
- 球员榜：保留射手榜、助攻榜、扑救榜、黄牌榜，使用简单 Tab 切换。
- 本地补充：保留港澳/内地直播入口、集锦入口、新闻标题来源外链，不抓正文。
- PWA：支持手机和 PC 浏览器访问，可作为网页分享。

## 常用命令

```bash
pnpm install
pnpm test
pnpm lint
pnpm build
pnpm dev
```

## 数据更新

赛程比分由 App 自动请求 ESPN scoreboard；直播/集锦链接、新闻、积分榜和球员榜仍保留本地精选补充。具体位置见 [数据更新指南](./docs/data-update-guide.md)。

## 发布前检查

准备发给亲友试用前，建议确认：

- 赛程、积分、球员榜和新闻外链已经更新到当前比赛日。
- 首页显示 `自动更新 · ESPN`，或在 ESPN 暂不可用时显示本地兜底状态。
- 每场比赛至少有一个直播或集锦入口。
- 港澳和内地观众常用入口都保留。
- `pnpm test`、`pnpm lint`、`pnpm build` 都通过。
- 手机宽度和 PC 宽度各打开一次，确认没有横向滚动或明显遮挡。

## 部署

当前项目适合部署到 GitHub Pages。内地访问速度不稳定时，可以另外准备一个镜像入口，代码不用分叉。具体操作见 [部署指南](./docs/deployment-guide.md)。
